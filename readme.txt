my press articles is the all in one wordpress plugin which includes database optimization feature (new), customize feature for wordpress login form, related posts feature, sticky post widget, social media widget, google map, google analytics and many more. 

== Description ==

Current features include 1) Google Analytics 2) Related Posts 3) Sticky Posts Widget 4) Social Media Widget and buttons in post content 5) Google Map with shortcode 6) Facebook Fan Page 7) Customize feature for wordpress login form. 8) Database optimization

== Frequently Asked Questions == 

= How to use this plugin? =
 
Follow the tutorial carefully and you will master how to use it within minutes.    

== Installation ==

1) Download my press articles to your blog and activated it.
